# UPI Transactions Analytics Dashboard | Power BI

An interactive Power BI dashboard built to analyze UPI transaction data, 
providing insights into transaction performance, payment trends, merchant activity, 
banking partnerships, and geographical distribution. The dashboard enables users to 
monitor KPIs and explore data through dynamic filters and visualizations.

# 📌 Features
📊 Total Transactions KPI
💰 Total Transaction Volume (INR)
✅ Success Rate Analysis
💵 Average Transaction Value
📈 Monthly Transaction Trends
🗺️ Transaction Distribution by City
🏦 Top Bank Partners by Transaction Volume
💱 Currency-wise Transaction Analysis
🛍️ Merchant Category Breakdown
🎛️ Interactive Slicers (Status, Gender, City, Merchant Name, Payment Method, Receiving Bank)

# 🛠️ Tools & Technologies
Power BI
Power Query
DAX
Data Modeling
Data Visualization

# Key Insights
Analyzed 20,000 UPI transactions.
Total transaction volume exceeded ₹19.87 Million.
Achieved an overall 80% transaction success rate.
Average transaction value of ₹993.61.
Identified monthly transaction trends and top-performing merchants.
Compared bank partners based on transaction volume.
Visualized city-wise transaction distribution using maps.
Enabled interactive business analysis through filters and KPIs.

# Project Objective

The objective of this project is to demonstrate how Power BI can 
transform raw UPI transaction data into meaningful business insights through 
interactive dashboards. It helps stakeholders monitor transaction performance, 
identify trends, evaluate merchant and banking performance, and support data-driven decision-making.
